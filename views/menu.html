Regras do Jogo

Aqui é infomado as regras

do jogo e como funciona:

1. Todos os alunos devem estar em um computador ou celular individualmente para participar.

2. O professor deve criar a atividade com suas configurações para mandar o código da atividade para os alunos.

3. O professor deve analisar as perguntas feitas pelos os alunos para colocar na atividade.

4. O aluno pode fazer no máximo 3 perguntas com 3 respostas onde 1 é verdadeira e 2 falsas.

5. Os alunos devem estar em grupos separados para que não aconteça trapaça.

6. Pensar em mais regras do jogo para ser aplicado...

Como funciona o jogo

1. O professor ou usuário cadastrado deve criar a atividade de acordo com suas configurações.

2. Após criar a atividade será gerado um código aleatório de convite para os alunos entrarem com seu nome e serem aceitos

3. pós os alunos serem aceitos o professor deve colocar qual aluno vai para qual grupo feito.

4. Após os alunos estarem em seus grupos eles devem criar as questões do jogo onde 1 resposta será correta e 2 serão falsas, max 2 questões por aluno.

5. Após, o professor deve aceitar ou recusar as questões dos alunos para ser colocado no jogo.

6. O professor deve começar o jogo selecionando um grupo para começar, a ordem é aleatória do jogador para jogar e responder as questões.

7. O aluno tem 15 segundos para responder às questões, caso o tempo passe ele tem a opção de ajuda do "Professor", "Grupo" e “Passar” de cada grupo.

8. Após as questões terminarem o jogo encerra mostrando os pontos e tendo um grupo vencedor.
